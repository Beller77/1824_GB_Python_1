Установлен ли пакет pillow в глобальном окружении:

PS C:\PycharmProjects\1824_GB_Python_1> pip freeze
PS C:\PycharmProjects\1824_GB_Python_1> pip show pillow
WARNING: Package(s) not found: pillow

Installing collected packages: pillow

Создать и активировать виртуальное окружение. Убедиться, что в нем нет пакета pillow:

(venv) PS C:\PycharmProjects\1824_GB_Python_1> pip freeze
(venv) PS C:\PycharmProjects\1824_GB_Python_1>

Установить в виртуальное окружение pillow версии 7.1.1 (или другой, отличной от самой свежей):

(venv) PS C:\PycharmProjects\1824_GB_Python_1> pip install pillow==8.1
Collecting pillow==8.1
  Using cached Pillow-8.1.0-cp39-cp39-win_amd64.whl (2.2 MB)
Installing collected packages: pillow

(venv) PS C:\PycharmProjects\1824_GB_Python_1> pip freeze

(venv) PS C:\PycharmProjects\1824_GB_Python_1> pip show pillow
